# 哈吉米AI 项目语境

本文件记录项目里的固定用词，防止后续开发和文档把同一件事说成多个对象。

## Language

**画布**:
本项目唯一的工作流编辑与生成工作台。所有画布开发、修复、验收和文档说明默认都指向这个画布。
_Avoid_: 新画布、旧画布、新版画布、旧版画布

**当前画布运行链路**:
画布在当前项目里的实际运行链路，包括现有前端入口、打包资产、桥接脚本和后端接口。它是当前开发对象，不是临时参考版本。
_Avoid_: 旧链路、旧运行时、旧画布链路

**已废止的独立画布重建方案**:
曾经尝试过的替代画布实现方案，已退出当前主线。它不是一个可与画布并列推进的产品或开发目标。
_Avoid_: 新画布、第二画布、新版画布
