# 贯穿三层的推断纪律（开工前必读）

> 第一层"看到什么"可观察、可取证。本 skill 的三层——意图、功能、质量——**都看不见,只能从选择反推**。这套纪律管着 L2/L3/L4 的每一步,是把"严谨深读"与"凭感觉脑补"分开的唯一分水岭。先装它,再做任何一层。

## D1. 意图从"选择 vs 备选"反推

推断意图的**唯一合法路径**:创作者选了 X,而本可以选 Y/Z;选 X(且大量选择 collectively 收敛到同一指向)才说明意图。

- **单个选择 = 弱证据;选择的收敛模式 = 强证据。** 光、色、置景、姿态全部指向"温暖居家",这个收敛本身是证据;只有一个暖色,不够。
- 写每个意图判断时,先问:"这个选择的备选是什么?为什么没选备选?"答得上来,意图才落地。

## D2. denotation → connotation（明示义到隐含义，Barthes）

第一层做的是**明示义**(denotation):中立的字面层,"这是什么"。本 skill 做**隐含义**(connotation):文化、情感、联想的层。两者天然衔接——L1 的事实是 L2 的原料。

- **去自然化(de-naturalize)**:照片的"真实感"会把构造出来的意义伪装成"自然/理所当然",商业图尤其如此。读意图 = 戳破这层:那个"显而易见"的意思,是被一组具体选择搭出来的。把 "obvious" 还原成 "constructed"。

## D3. 三样东西务必分开，别塌缩成一个

| | 是什么 | 怎么处理 |
|:--|:--|:--|
| 画面 **carries** 的意义 | 靠文化代码可读,创作者没意识到也可能存在 | 用代码读出来,标"画面携带" |
| 创作者 **intended** 的意图 | 从选择模式 + 类型惯例推断 | 带置信度推断,标"意图" |
| 观看者 **projects** 的联想 | 图像是多义的(polysemy) | 列为多义可能,标"可能的解读" |

**红线:绝不把"图像意味着 X"等同于"创作者意图 X"。** 意义可以超出意图而存在。这是头号过度解读陷阱。

## D4. 四档归因（每个意图判断都过一遍）

判断某个效果是不是"意图"前,归一次类:

1. **作者刻意(authored)** ← 只有这档算个体意图
2. **媒介/约束的副产品(incidental)** ← 手机抓拍的杂乱背景不是"刻意的亲密"
3. **类型默认(genre default)** ← 这个品类本来就这样,不算个体意图(如所有电商图都白底)
4. **我在投射(projection)** ← 观看者自己的联想

只有 ① 进"意图";②③④ 各自标注,不混入意图。

## D5. 置信度校准 + 存疑清单（每层都做）

每个核心判断挂一档置信度:

- **强烈指向**:多个独立选择收敛到同一结论
- **可能**:单一或模糊线索
- **无法判定**:证据不足,或证据互相矛盾

每一层结束时,附**该层的存疑清单**:哪些读法是多义的(可 X 可 Y)、哪里判不了、哪里可能在投射。三层的存疑清单层层累积,汇入最终输出。

## D6. 判断的标准形态（推断版的"取证"）

第一层的每句描述挂"视觉依据"。本 skill 的每句判断挂"它依据的选择 + 置信度":

> **判断 = 结论 + 它依据的画面选择/描述事实 + 置信度**

例:"意图营造'温和可信',依据:柔光 + 去饱和高级灰 + 主体托产品的珍重姿态 + 天然棉麻材质,四项收敛;置信度高。"

## D7. 跨层数据流:L1 的矛盾清单 → L4

第一层做一致性自检时会列"物理矛盾清单"(光向↔投影对不上、遮挡成环、材质↔高光不符等)。**这份清单一路喂到第四层 J2(执行质量)和 J4(伪影/合成风险)**——物理矛盾要么是执行失误,要么是合成/AI 痕迹。读到 L1 的存疑清单时,标记好,留到 L4 用。

## D8. 三层的边界(别 double-count)

四层各管一段,缝要清楚:

- **L2 意图 = 内部效果**:想让你*感受/看见/相信*什么。
- **L3 功能 = 外部任务**:想让你在现实里*做*什么、给*谁*、在*哪*出现。
- **L4 判断 = 任务完成度**:上面那个任务,*做得好不好*。
- **缝上的(观看位置 / address 模式)**:在 L2 读"被摆放的姿态",在 L3 读"为了什么行为"。同一现象,两层各取一面,不重复计。

装好以上八条,进入 [layer2-intent.md](./layer2-intent.md)。
