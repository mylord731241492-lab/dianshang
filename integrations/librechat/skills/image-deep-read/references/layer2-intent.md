## B1. 置景与道具：内容即意图 ★

> 这是本层的重心。**镜头前每一个被摆放的东西都是表意选择**(mise-en-scène);场面调度甚至把"什么缺席"也算作有意的选择。逐项追问"为什么是它、它 connote 什么":

- **选角 / 主体选择(casting)**:主体是谁——年龄/类型/长相/气质/表情,被设定来代表谁、让谁代入、让谁向往。产品图:选了哪个 SKU、哪个角度、满瓶还是用过。
- **场景 / 地点(setting)**:在哪,以及这个地点借来什么联想(阳光厨房=居家健康;水泥 studio=clinical 现代;自然风景=纯净自由)。真实场景借"真实感",搭景借"控制力"。
- **道具 / 物件(props)**:每个物件都在干活——暗示生活方式/阶层/季节/功能/叙事。一本书、一杯咖啡、一枝特定的花各带代码。Barthes 的经典例:番茄经由 metonymy 指代"意大利性"。**包括刻意的缺席**(空出本该有东西的位置)。
- **造型 / 服化(costume/makeup)**:服装/发型/妆 signify 角色/阶层/年代/情绪/目标身份。
- **动作 / 姿态 / 视线(staging & gesture)**:主体在做什么、怎么拿产品、**视线方向**(看镜头=直接对"你"说话;看别处=被观察/向往/叙事中)、肢体语言、人物间的距离与权力关系。姿态是意图密度最高的地方之一。
- **隐含叙事(implied narrative)**:这一帧暗示了故事里的哪个时刻——之前/之后、一段关系、一个处境。冻结的瞬间在讲什么故事?
- **世界一致性(world coherence)**:所有摆设是否 collectively 搭出一个自洽的世界(年代/季节/社会语境/美学宇宙)?一致,还是刻意的违和?

每一项都要落到 D1:这个置景/道具/姿态的备选是什么?选它而非备选,指向什么意图?

## B2. 形式技术的意图（cinematography 侧）

第一层记录了色/构图/光/镜头**是什么**;这里问**为什么是这个而非备选**,每项形式选择买到了什么:

- 色彩选择 → 情绪/品类代码(去饱和高级灰=premium 克制 / 高饱和=活力或廉价)
- 构图选择 → 引导/权力/稳定(居中=庄严对峙 / 三分=自然动态 / 仰拍=高大权威)
- 光选择 → 情绪/真实性(硬光=戏剧紧张 / 柔光=温柔梦境 / 顶光底光=反常)
- 镜头/景深选择 → 注意力/疏离(浅景深=聚焦隔离 / 大景深=信息在场纪实)

每项配"它服务于什么 + 备选会怎样"。

## B3. 风格谱系与互文

- **风格定位**:这张图说的是哪种视觉语言(editorial 时尚 / 纪实 / 电商产品 / 纯艺 / 品牌 KV / UGC 素人感)。类型自带默认意图与阅读惯例——注意这是 D4 的"类型默认",别把类型惯例误记成个体意图。
- **互文 / 风格借用(intertextuality)**:是否刻意 evoke 另一种类型/年代/作者/名作来借它的联想(把广告拍成 Wes Anderson 的对称糖果色、拍成文艺复兴油画的光、套 90 年代胶片质感)。意图 = 进口被引用风格的 connotation。互文是个体意图(选择借哪个),要点出。

## B4. 情绪与氛围的制造

前面所有选择 collectively 制造的那个情绪。但**必须写成"由[这些具体选择]共同制造[什么情绪]"**,不是空贴"温暖/疏离"。情绪判断要落到产生它的选择上——这正是 D1 的收敛:多个选择都指向同一情绪,情绪判断才成立。

## B5. 观看位置与真实性姿态（通往 L3 的缝）

- **观看位置 / address 模式**:图像把观看者摆在什么位置?直接 address("你",被召唤)还是旁观?向往式 framing 把你摆成"欲求者"。注意 Barthes 的"锚定"(anchorage):画面里的文字常把观看者往一个读法上推(引导你避开某些含义、接收另一些)。
- **真实性姿态**:图像*想显得*自发/抓拍,还是精致/向往?"UGC 素人感"本身常是被制造的真实(manufactured authenticity)。"看起来多 made 还是多 caught"是个意图选择。
- **边界**:这一组踩在 L2/L3 缝上(D8)——这里只读"被摆放的观看姿态";"为了什么真实行为"留给 L3。
