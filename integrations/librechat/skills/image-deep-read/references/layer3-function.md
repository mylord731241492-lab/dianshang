# L3 · 功能层（它在现实里干什么活）

> 回答:这张图在现实世界**承担什么任务**。这是外部任务层。master variable 是**语境**——它统摄其余维度,而且语境常常单凭图判不出来,未知时整层写成**条件式**。

## 一个必须先立的认识:语境是框架,不是维度之一

同一张肖像,可以是广告、editorial、个人照、图库图、政治海报——功能完全不同。所以**语境不是四个维度之一,是统摄受众/信息/行为的框架变量**。

- **语境已知**(知道出处/投放位):正常推断功能。
- **语境未知**(只拿到一张没出处的图):功能分析必须写成**条件式**——"若是电商主图,则…;若是品牌社媒,则…;若是 editorial,则…",不要武断断言单一功能。这是诚实做法。

---

## 理论锚:Jakobson 传播六功能（替换粗糙的"信息/行为"二分）

一条信息很少只有一个功能;**识别主导功能 + 辅助功能**(诸功能不互斥,通常一个主导)。六功能映射到图像:

| 功能 | 朝向 | 在图像里 |
|:--|:--|:--|
| **指称** referential | 语境/对象 | 传递关于对象的事实(产品规格、是什么、数据) |
| **情感** emotive | 发送者 | 表达品牌/发送者的态度与个性(品牌声音) |
| **意动** conative | 接收者 | 引导接收者行动(CTA、买、点) |
| **交感** phatic | 接触/通道 | 维持接触与在场(刷存在感的内容,信息量极低但维系关系) |
| **元语言** metalingual | 代码 | 关于"怎么读"(信息图图例、用法说明、教你解码) |
| **诗性** poetic | 信息本身 | 形式本身即目的(craft/美就是重点,KV、slogan) |

> 很多品牌社媒内容主导功能是**交感**(维持在场),不是意动(直接卖);很多 KV 主导是**情感+诗性**(建立品牌感觉),意动只是辅助。别把所有商业图都默认成"卖货"。

## 维度

- **F1 主导功能 + 辅助功能**:六选,标出主导,列出辅助。每个判断挂依据(从画面什么特征读出这个功能)。
- **F2 目标受众**:给谁看。但受众常需品类/语境知识——单凭图,往往只能**从代码反推画像**(用了什么符号、什么阶层暗示、什么生活方式)。反推到的画像标注置信度,别当确证。
  - 提醒:推断受众时倾向"广泛共鸣"的读法,避免不必要地窄化(如轻易锁死成某个狭窄人群标签);窄化既可能误读意图,也是 L4 的文化/排他风险点。
- **F3 语境关系(★ master variable)**:出现在哪(门店/信息流/详情页/包装/财报/户外大屏)。语境改变功能,也改变 L4 的成功标准。**未知则条件式**(见上)。
- **F4 行为引导的落点**(意动的具体化):希望你做什么(点/买/分享/停留/信任/记住),以及**画面如何 afford 这个行为**(CTA 位置、视线导向、留白引导、最亮点是不是落在产品/按钮上)。没有 CTA/文字的图,往往不是硬转化资产,而是品牌/记忆资产——这点直接影响 L4 怎么判。
