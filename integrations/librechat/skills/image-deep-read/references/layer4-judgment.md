# L4 · 判断层（做得好不好）

> 回答:这张图**能否完成它的任务**。这是最后一层,站在前三层证据上。master variable 是**适配度**——**质量 = 对(L3 推断出的)功能的适配度,不是绝对美学**。一个不挂功能的判断,就是"漂"判断。

## 两条定盘的纪律

**纪律一:先声明标准,再下判断(Feldman)。** 判同一张图,用写实标准 / 形式标准 / 情感标准 / 功能标准,结论会变。本框架默认**功能性标准**(它能否完成 L3 那个任务),但要声明这是个选择,不是唯一真理。

**纪律二:平衡 ≠ 软,且对事不对人。**
- balanced 意思是**完整**:好处和问题都说、都有证据、都给可操作建议。不是"总找点好话敷衍"。
- 判断针对 **artifact 的适配度,不针对创作者**。指出执行问题不是对人的否定。
- 区分**客观可评**(可读性、技术缺陷)与**可争议/品味**(差异化、情绪效力)——标清哪个是哪个,别把个人品味当普世结论。

## 维度

### J1. 适配度（master）
视觉呈现是否服务于 L3 推断的功能?逐条说**哪里服务、哪里背离**。这是质量的核心定义,其余三项都是喂给它的输入。

### J2. 执行质量（挂着媒介的 checklist）
- **可读性**:文字/CTA 在它**实际被看到的尺寸和语境**下读得清吗?(信息流缩略图 vs 户外大屏,标准不同)
- **技术缺陷**:偏色、banding、合成接缝、抠图穿帮、透视错误、伪影、压缩噪点。
- **该锐的地方锐不锐**、对焦平面对不对。
- **接入 L1 的矛盾清单(D7)**:第一层列的物理矛盾(光向↔投影、遮挡成环、材质↔高光不符)直接喂这里——要么是执行失误,要么是合成痕迹(后者去 J4)。

### J3. 差异化（需要参照集）
在同品类里有没有辨识度——但这**需要品类视觉惯例/竞品知识**,单凭一张图判不了。
- 有竞品集 → 对比着判。
- 没有 → 从类型惯例推,并明确标"此判断需竞品集确认"。
- 常见陷阱:一个品类用滥的视觉语言(如美妆的"晨光+天然+珍重托瓶")会拉低差异化,即使单看执行不错。

### J4. 风险信号（系统化 + 接入真实合规）
分六类排查:

1. **可信度 / 合规风险**(★ 接入现实法规):伪科学感、过度承诺。**美妆/健康品类有硬红线**(中国现行):
   - 化妆品广告**不得明示或暗示医疗作用、不得用医疗用语**(治疗、抗菌、消炎、祛疤、生发、瘦身、各类皮肤病名等);
   - **功效宣称须与注册/备案一致并有科学依据**;普通化妆品**不得宣称祛斑美白、防脱发等特殊功效**;
   - **量化宣称**(如"28 天淡纹""7 天淡斑")须连同**生效时限一并举证**;
   - 禁绝对化用语(最/第一)及"纯天然/有机/食品级/可食用";
   - **画面层面同样算数**:暗示医疗场景、做疗效前后对比图、暗示"全人群适用"一样踩线;制造"容貌焦虑"的营销内容已被明令禁止。
   - 注意:**图像本身可能干净,风险常在后续文案/宣称层**——若图要配 efficacy 文案,这些红线全部适用。判断时把"图 + 可能的文案"一起看。
2. **反感 / 疲劳风险**:促销压迫、元素堆砌、硬卖、banner-blindness。
3. **空洞风险**:空洞高级感——有风格无信息、没 message;漂亮但换个品牌也成立(和 J3 的"用滥语言"互相喂)。
4. **伪影 / 恐怖谷风险**:AI 悬浮感、合成感、uncanny valley(L1 矛盾清单喂这里)。
5. **文化 / 敏感风险**:刻板印象、文化错配、排他性(对接 F2 的"避免窄化")。
6. **法律 / IP 风险**:未经许可使用他人享有著作权的美术作品会构成侵权;肖像权;虚假宣传 / 反不正当竞争。

> 合规与法规会变、且因地区/品类而异。**本清单是排查触发器,不是法律意见**;命中红线时标记并建议核实当时当地的现行规定,不要下"绝对合规/绝对违规"的断言。

### J5. 判断的置信度 + 反例
每个判断带置信度,并说明"**什么会改变它**"(如:差异化判断会因拿到竞品集而改变;合规判断会因补上文案而改变)。客观项与品味项分开标。
