# 输出汇总 · L2 → L3 → L4 + 置信度/存疑

> 把三层装进固定结构,**每个核心判断带依据 + 置信度**,结尾汇总贯穿三层的存疑清单。一份合格的深读,末尾一定有"哪些存疑、哪些判不了、哪里可能在投射"。

## 完整示例（接续描述层的同一张虚构图）

> 用 `image-reverse-describe` 输出契约里那张**同一张虚构窗光护肤肖像**,演示从描述到深读的接续。仍是虚构图,只演示推断的颗粒度与置信度写法。

**# 深读:一帧暖窗光护肤肖像,意在以"温和·天然·日常仪式"建立品牌信任;执行适配度高,但差异化偏弱、空洞风险中等**

**## 前置:描述层依据**
取自 L1:4:5 竖幅;暖侧光(左前 45°柔光)、伦勃朗光;主体侧身托乳白护肤瓶(实焦、占 55%);前景虚化叶作框景;背景去饱和灰青室内;暖橙↔灰青低强度冷暖对比;皮肤次表面散射、棉麻哑光、瓶身材质介于磨砂玻璃/哑光塑料;瓶身标识不可辨。**L1 矛盾清单:无重大物理矛盾**(光向↔投影自洽);仅瓶身材质与冷补光来源两处存疑。

**## L2 · 意图**
- **视觉动线**:先落在受暖光的脸,顺前臂线条滑到乳白瓶身(画面最亮块)——被设计过,非偶然。
- **置景与道具**:
  - 选角:年轻、肤质好但不浓妆的女性——代表"可向往又可代入"的理想日常使用者。备选(浓妆/中年/男性)都不选,指向"自然日常护肤"的定位。
  - 场景:暖光室内(窗光)——借"居家/晨间/健康/平静"的联想。备选(实验室/纯色棚)会偏向"科学/clinical",这里反向选了"生活感"。
  - 道具:护肤瓶被珍重托于胸前=仪式/呵护;前景一枝叶=天然/新鲜。"刻意的缺席":没有价格、没有促销标、没有成分图——刻意不走硬卖。
  - 造型:哑光棉麻=朴素/低调奢华(premium-casual)。
  - 姿态/视线:托瓶姿态把产品举到视觉强势位;视线读法存疑(看向瓶=引导你看产品 / 看向别处=私密仪式的旁观感),两种读法分别导向不同 address(见下)。
  - 隐含叙事:一个安静的晨间护肤瞬间。
  - 世界一致性:暖光+天然材质+朴素造型 collectively 搭出"自然·平静·日常"的自洽世界。
- **形式技术意图**:去饱和高级灰+暖柔光=premium·克制·温柔的品类代码(备选:高饱和硬光会偏活力/廉价);浅景深隔离主体=聚焦+亲密。
- **风格谱系与互文**:品牌 KV,"clean/natural/晨光 wellness"谱系;隐约互文"晨光生活方式"美学。注意"晨光+天然"是该品类**类型默认**(D4),不全是个体独创意图。
- **情绪制造**:由[暖柔光 + 去饱和 + 珍重托瓶姿态 + 天然棉麻] 四项共同制造**平静、温暖、可信、轻度向往**。
- **观看位置/真实性姿态**:偏向往式/旁观式(你在看一个私密仪式→"这可以是你");姿态是"精致但想显得自然"的被制造的平静感。
- ▶ **意图陈述**:通过上述收敛的一组选择,图像意图让观看者感到"此产品属于一种平静、健康、天然的日常仪式,且温和可信"。**置信度:高**(四组以上独立选择收敛)。
- ▶ **意图存疑**:视线方向不明→"直接向往"还是"叙事旁观"判不定;品牌的具体差异化主张,单凭图无法读出。

**## L3 · 功能**
- **语境判定**:出处未知→**条件式**。
- **主导+辅助功能**:
  - 若为**品牌 KV/社媒**:主导=**情感+诗性**(建立品牌感觉与在场),辅助=意动(轻推认知)。依据:无 CTA/文案、craft 前置、情绪主导。
  - 若为**电商主图/详情页**:主导转为**指称+意动**(展示产品、推动加购),此构图的"产品不够大、标识不可辨"会成为弱点(见 L4)。
  - 若为**杂志 editorial**:主导=**诗性**。
- **目标受众**(从代码反推):追求"温和/天然/克制"定位的护肤消费者;生活方式偏向都市、注重质感。**置信度:中**(代码一致但缺语境)。倾向广泛共鸣读法,不窄化到单一狭窄人群标签。
- **行为引导落点**:乳白瓶身是动线终点+最亮块→afford"产品识别与记忆";无 CTA→更像品牌/记忆资产,非硬转化资产。
- ▶ **功能陈述**:最可能是品牌情绪/记忆资产(情感+诗性主导);**置信度:中,条件依赖投放位**。

**## L4 · 判断**
- **声明标准**:采用**功能性标准**(能否完成"建立平静·天然·可信的品牌感觉并让产品被记住");注明这是个选择。
- **J1 适配度(master)**:**高**。几乎每个选择(暖柔光/去饱和/珍重姿态/天然材质/产品作最亮动线终点)都收敛到"平静·温和·可信"的目标效果,内在一致。
- **J2 执行质量**:技术干净(L1 一致性自检通过、无物理矛盾);唯一执行短板=**瓶身标识不可辨**——若产品识别重要(电商场景)则是问题,品牌情绪场景下可接受。
- **J3 差异化**:**偏弱**。"晨光+天然+珍重托瓶"是美妆品类用滥的视觉语言。**此判断需竞品集确认**;按惯例推,属"执行好但familiar"。
- **J4 风险信号**:
  - 可信度/合规:**图像本身低风险**(无可见宣称)。**但**若配上 efficacy 文案,美妆红线全部适用——不得用医疗用语、功效须匹配备案、量化宣称(如"X 天淡纹")须连时限举证、不得制造容貌焦虑。**把图+拟用文案一起评估**。
  - 空洞风险:**中等**——很美很平静,但是否说出了"这个品牌/产品"区别于任何 clean-skincare 品牌的东西?(与 J3 互相喂)
  - 反感/伪影/文化:均低;文化上注意别把隐含使用者窄化。
  - 法律/IP:核对前景叶等素材授权、模特肖像授权。
- **J5 置信度 + 反例**:适配度判断高;差异化判断中(拿到竞品集会变);合规判断条件性(补上文案会变)。
- ▶ **整体判断**:在功能性标准下,作为品牌情绪资产**适配度高、执行干净**;主要短板是**差异化弱 + 空洞风险**(漂亮但可能换个品牌也成立),主要可操作改进:注入一个该品牌专属的视觉记忆点(独有道具/色/光语言),并在配文案时守住美妆合规红线。

**## 贯穿三层存疑与置信度汇总**
- 合并存疑:视线方向(L2)、投放语境(L3)、竞品集缺失(L4-J3)、文案未定故合规待评(L4-J4)、瓶身材质(L1)。
- **最该补证据的 3 点**:① 投放语境(决定 L3 功能与 L4 标准);② 竞品集(决定差异化判断);③ 拟用文案(决定合规风险)。这三点不补,本深读的功能与判断部分都是条件性的。

---

> 注意这份深读如何处处挂着"依据 + 置信度",并在结尾把**最不确定、最该补证据的点**单独拎出——这就是 inference-discipline 的 D5/D6 在输出层的兑现:深读不是给结论,是给"有多少把握的结论,以及凭什么"。
