## §A 构图

构图判定的通病是**只贴标签**（"这是三分构图"）却不给依据。本步要求：每个构图判断都落到"画面里什么落在什么位置"。

### 要判定的内容

1. **构图法**：从下面清单里命名，但**必须配依据**。
   - 居中 / 对称（双侧对称、径向对称）/ 三分 / 黄金分割·螺旋 / 对角线 / 三角形（稳定感）/ 引导线式 / 框架式（框景）/ 重复阵列 / 大量留白 / X 形 / S 形（曲线引导）。
   - 取证示范：`判为三分构图。依据：主体落在右三分线 (x≈0.66)，地平线压在下三分线 (y≈0.66)，二者交于右下交点——两条强势元素同时落位三分线，非偶然。`
2. **视觉重心位置**：给坐标 + **成因**（是什么把视线吸过去的：最高明度 / 最大对比 / 最大面积 / 朝向汇聚 / 唯一暖色）。注意区分"几何中心"与"视觉重心"——二者常不重合。
3. **引导线（leading lines）**：给**实际向量**——从哪到哪、指向何处（"道路从左下 (0.1,0.95) 收向灭点 (0.58,0.45)，把视线推向主体"）。
4. **平衡**：对称 / 不对称（动态平衡）/ 失衡。依据＝视觉重量的左右、上下分布。
5. **负空间（negative space）**：占比 + 位置 + 作用（留白让主体透气 / 放标题）。
6. **裁切（crop）**：主体是否被画框切断、切在哪、是否刻意（如人物切在关节处 vs 留出引导空间 lead room）。

### 精度要求

- 构图法不可空贴标签，一律"标签 + 落位依据"。
- 引导线给起止坐标与指向。
- 视觉重心给坐标 + 成因。

## §B 文字与符号

画面里的可读信息层。**核心诚实原则：能确认才命名，认不出就只描述形态。**

### 要清点的内容

1. **可读文字**：
   - **逐字转录**（若清晰可读）；模糊处标"部分不可辨"。
   - **位置**：坐标。
   - **字体特征**：衬线 / 无衬线 / 手写体 / 显示体（display）；字重（细/常规/粗/特粗）；是否斜体、是否大写。
   - **字号层级**：主标题 / 副标题 / 正文 / 注脚——按相对大小与位置分级。
   - **语言/字符集**：中 / 英 / 日 / 韩…（若可辨）。
2. **Logo / 标识 / 品牌符号**：
   - 描述其**形态**（图形构成、颜色、位置）。
   - **品牌识别要克制**：只有形态足够清晰、确能认出时才写品牌名；否则写"某品牌标识，形态为 X，品牌不可辨"。**绝不臆造品牌名**——这是反推可信度的红线。
3. **符号 / 图形元素**：箭头、图标、装饰线、几何图形等，给位置 + **功能**（分隔 / 指示 / 装饰 / 信息编码）。

### 取证：文字视觉层级

文字也有"先读到谁"的层级。依据＝字号最大 / 对比最高 / 位置最强势者最先被读到。例：`文字层级：最大字号的"XXX"(上中, y≈0.15) 为视觉第一落点；其下小字为副信息；右下角注脚字号最小、最后读到。`

### 精度要求

- 可读文字尽量逐字转录 + 给位置 + 字体特征 + 层级。
- 品牌/标识严格区分"已确认"与"不可辨"。
- 符号给功能，不只给形状。
