# 步 1 · 空间结构与深度分层

> 这是描述层的骨架。目标：把画面拆成**深度层**（前/中/后景）与**显著性层**（主体/陪体/环境）两条正交轴，精确到方位、深度、占比、遮挡，并为每一个深度判断**挂上视觉证据**。

执行五小步：判定深度结构类型 → 逐层清点 → 遮挡与空间关系 → 深度线索取证 → 主体定位。

---

## 1. 先判定深度结构类型（决定用几层）

**不要对每张图都硬塞"前中后"三层**——会制造虚假精度。先归类：

| 类型 | 特征 | "层"怎么理解 |
|:--|:--|:--|
| **平面图形型** | 海报、信息图、纯设计版面，无空间纵深 | "层"＝设计元素的 **z-order 叠放**（背景块→图形→文字），不是物理远近 |
| **单层 / 浅景深型** | 微距、平铺 flat-lay、证件照、纯色背景人像 | 本质就一个面，或主体面+模糊背景两层，不要硬拆三层 |
| **标准三层型** | 有明确前/中/后纵深的常规场景 | 经典前景 / 中景层 / 背景 |
| **纵深连续型** | 风景、街景、大景深建筑 | 可能有 5 层以上；按自然的纵深断点分层，不强凑成三 |

**输出**：写明类型，写明"本图用 N 层"，写明"前中后景按物理纵深 / 按设计 z-order 理解"。

---

## 2. 逐层清点（每一层填满五项）

对每个深度层，填齐这五项——缺一项，可逆性就漏一块：

1. **内容**：具体到可辨识。"穿驼色风衣、侧身向右的女性"，**不是**"一个人"。
2. **方位**：用坐标系。`x≈0.4, y≈0.5，约在正中偏左`。元素跨区给范围。
3. **深度**：该层的相对距离（更近/更远），若线索允许给绝对估计（如"约 2–3 米外"）。**深度判断必须配第 4 步的取证**。
4. **处理**：对焦状态（实焦 / 虚化程度）、受光 / 背光、清晰度、有无动态模糊。
5. **占比**：该层 / 该元素约占画面面积的百分比。

> 一层一段，例：
> `前景（占底部约 25%）：横贯画面的金属栏杆 (y≈0.7–1.0)，强虚化。处理：失焦、偏暗。深度：最近层，距镜头最近。`

---

## 3. 遮挡与空间关系（让描述可重建的关键）

光有"每层是什么"还不够，要写清"它们怎么搭在一起"：

- **遮挡序（z-order）**：谁在谁前面。写成 `A 遮 B 遮 C`。这是确定前后关系**最硬**的证据。
- **框景元素 / repoussoir**：有没有前景元素（门框、枝叶、栏杆、虚化人影）框住或引向主体。这是一种特定的前景功能，要单独点出。
- **元素间相对方位**：`A 在 B 的左上方`、`A 与 C 部分重叠`、`D 紧贴画面右缘`。让任意两个关键元素的相对位置都明确。

> **遮挡成环＝矛盾信号**：如果出现 A 遮 B、B 遮 C、C 又遮 A，这在物理上不可能，记入存疑清单（可能是拼贴/生成破绽，但本层只标记不下结论）。

---

## 4. 深度线索取证（本步的严谨锚点）

**每一个深度层级的判定，都要列出是哪些线索支持的。** 这是把"我觉得这是背景"变成"判为背景，依据：……"的弹药库。二维平面之所以能读出纵深，靠的就是下面这组**单眼/图像深度线索**（源自视知觉研究中的 pictorial depth cues）：

| 线索 | 怎么读 | 它给出什么深度信息 |
|:--|:--|:--|
| **遮挡 / 交叠**（occlusion / interposition） | A 挡住 B → A 更近 | **最可靠**，但只给前后顺序（序数），不给距离 |
| **相对大小**（relative size） | 同类物体，看着小的更远 | 需"同类 / 已知尺寸"前提 |
| **熟悉尺寸**（familiar size） | 已知真实大小的物体，按其视角大小反推距离 | 绝对距离估计的锚 |
| **线性透视**（linear perspective） | 平行线向灭点收敛，越收敛越远 | 配合灭点给出纵深方向 |
| **纹理梯度**（texture gradient） | 同种纹理，越远越密、越细 | 连续地面/墙面的纵深 |
| **空气透视**（aerial / atmospheric perspective） | 越远越灰、越偏蓝、对比越低、越柔 | **远景判定的主力** |
| **高度位置**（height in field） | 地平面上，越远的物体在画面里越靠上（到地平线为止） | 地面物体的纵深 |
| **光影与投影**（shading / shadows） | 形体明暗与接触阴影暗示前后、离地高度 | 辅助，定位元素相对位置 |
| **虚化梯度 / 景深**（defocus / DOF） | 越虚＝越偏离对焦平面 | 摄影特有，标出"对焦平面"在哪个深度 |

> **关于虚化的重要细节**：模糊只告诉你"离对焦平面有多远"，**不告诉方向**——一个元素发虚，可能因为它比对焦面更近，也可能更远。要靠遮挡/相对大小等其他线索来判定它到底在前还是在后。别把"虚"直接等同于"远"。

**取证写法示范**：
```
此层判为背景。
依据：① 空气透视——楼群去饱和、偏蓝灰、对比明显低于中景；
      ② 相对尺寸——同为建筑，远小于可参照的近处结构；
      ③ 高度位置——位于画面上部 y≈0.10–0.30，贴近地平线；
      ④ 遮挡——下缘被中景人物肩部遮住。
四条线索一致指向"最远层"。
```

---

## 5. 主体定位（把两条轴合上）

到这里才处理"主体"——而且是把**显著性轴**接到**深度轴**上：

- **指认主体**：画面里视觉重量最大、最先被注意的元素。
- **说清它落在哪个深度层**：主体可能是前景（贴镜头），也可能在中景层而前景是虚化框景物。明确写"主体位于中景层"。
- **说清它为何读作主体**——列依据，通常是这几样的组合：唯一实焦 / 最大占比 / 最高对比 / 位于视觉强势位（三分交点、正中）/ 被光打亮 / 视线或朝向引导 / 色彩跳出。
- **标出陪体与环境**：陪体＝次要但有叙事/陪衬作用的元素；环境＝前景+背景里纯做衬托的部分。各自一句话点明角色。

> **示范**：
> `主体＝侧身女性，位于中景层 (x≈0.4, y≈0.5)。读作主体依据：全图唯一实焦区 + 占画面约 30%（最大单体）+ 落在左三分交点 + 暖光打亮使其从冷调背景中跳出 + 面部朝向画面纵深引导视线。陪体＝其手中产品（次焦点，引导视线终点）。环境＝虚化栏杆（前景框景）+ 去饱和楼群（背景衬托）。`
