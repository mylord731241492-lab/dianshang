# 步 2–4 · 表面三维度：光线 / 色彩 / 材质

> 这三者**物理耦合**：光是因，色彩和材质是被光揭示的果。所以判定顺序固定为 **光线 → 色彩 → 材质**。三者都套同一纪律：每个判断从可见痕迹反推，给出依据和可复现的量。本文件较长，按需跳读。

## §A 光线

光会在画面里留下物理痕迹（投影、高光、明暗交界线），所以光线是**最能严谨反推**的维度——你不是在猜光，是在从痕迹里读光。

### 要判定的六要素

逐项给判断 + 依据 + 精度：

| 要素 | 从哪些可见痕迹反推 |
|:--|:--|
| **光源方向** | ① 投影指向**背离**光源；② 受光面亮、背光面暗；③ 明暗交界线（terminator）大致**垂直于**光照方向。综合三者定方位。 |
| **光质（硬/软）** | 看投影边缘：**锐利清晰=硬光**（小或远的光源，如裸灯、正午太阳）；**柔和渐变=软光**（大或近的光源，如柔光箱、阴天天光）。高光形态同理：硬光高光小而亮，软光高光大而柔。 |
| **色温（暖/冷）** | 看高光与受光面的色偏：暖光→暖高光 + 偏冷阴影；冷光反之。天空光/环境补光通常偏冷。 |
| **光比 / 反差** | 受光面到阴影面的明度跨度：跨度大=高反差/硬调；跨度小=低反差/平光。可粗估亮暗比。 |
| **光源数量** | 数清几套投影方向、几组高光：多套互不平行的投影方向 = 多光源或有补光（主光 key + 补光 fill + 轮廓光 rim）。 |
| **光型（人像专用）** | 见下表。 |

### 人像/主体打光的经典光型（供命名 + 取证）

- **方位型**：顺光（正面平打，立体感弱）/ 侧光（45° 侧，立体感强）/ 逆光（光从被摄体后方来，前面暗、边缘亮）/ 轮廓光 rim（仅勾边）/ 顶光（自上而下，眼窝鼻下出影）/ 底光（自下而上，反常诡异感）。
- **经典布光**（从面部阴影形态反推）：
  - **伦勃朗光 Rembrandt**：暗侧脸颊出现一个倒三角受光区（颧骨下方的"伦勃朗三角"）。
  - **蝴蝶光 Butterfly**：光从正前上方来，鼻子下方投出蝴蝶/领结状对称阴影。
  - **分割光 Split**：光从正侧 90°，半脸亮半脸暗。
  - **环形光 Loop**：鼻翼一侧投出一小段弧形鼻影，不与脸颊阴影相连。
- **取证**：从**鼻影位置/形状、颊部三角的有无、眼神光（catchlight）在瞳孔的钟点位置**反推光型与光源高度。

### 精度要求

- 光源方位用方位 + 钟点描述："左上约 10 点钟方向、略高于主体视平线"。
- 给明暗交界线的走向、画面最亮区与最暗区的坐标。
- 例：`主光判为左前 45°、高于主体的硬光。依据：投影向右下、边缘锐利；颧骨下见伦勃朗三角；眼神光在瞳孔 10 点钟位。色温偏暖（受光面偏橙、阴影偏青）。另有一束偏冷弱补光自右后勾出发丝轮廓（右侧发缘见冷调高光），光比约 1:4（暗侧明显压暗但留细节）。`

## §B 色彩

色彩描述要做两件常被忽略的事：**量化**（保证可复现）与**区分固有色 vs 光染色**（保证不被光骗）。

### 要判定的内容

1. **色板**：主色 / 辅色 / 点缀色（accent）。每个给 **HSL（色相°、饱和度%、明度%）或近似 Hex 范围** + **描述性命名**。
   - 量化与描述**并存**："釉质暖橙 `H≈25° S≈70% L≈55%`"。
   - **不要只给干色名**（"橙色"信息量太低，且不可复现）。也不要只给十六进制（丢了质感）。两者一起给。
2. **色调倾向**：暖 / 冷 / 中性，**配依据**。
3. **对比关系**：互补 / 邻近（类似色）/ 分裂互补 / 明暗对比 / 冷暖对比。**指出具体是哪两处的哪两色在对比**。
4. **饱和度分布**：整体高饱和 / 低饱和（高级灰）/ 分区（如主体高饱和、背景去饱和）。
5. **色彩的空间分布**：哪个区域是什么色——回接坐标系与深度层（如"暖橙集中在中景主体，青蓝铺满背景"）。

### 取证：固有色 vs 光染色

同一物体在暖光下会整体偏暖。判断颜色时必须说清：这是物体**本身的固有色（albedo）**，还是**被光源染上的色**？
- 取证示范：`整体判为暖调。依据：① 光源高色温（夕阳/钨丝特征）；② 肤色、白墙均带橙——白墙本应中性，其偏橙说明是光染而非固有；③ 阴影里也渗暖反射。注意：背景植物的绿为固有色（受光弱处仍绿），不是光染。`
- 对比关系示范：`互补冷暖对比——中景主体的暖橙 (H≈25°) 对背景的青蓝 (H≈205°)，二者色相近 180° 对置，是画面张力的主来源。`

### 精度要求

- 主色至少给到 HSL 三档或 Hex 范围。
- 对比关系必须落到"画面哪两处"，不能只说"有冷暖对比"。
- 区分固有色与光染色，至少对主体和最大背景面做这个区分。

## §C 材质与纹理

**材质是从光学行为反推的**——不是认出"这是金属"，而是读出"这个表面的反射/透射/散射行为像金属"。用 PBR 的眼睛看（镜面反射 specular / 漫反射 diffuse / 次表面散射 SSS / 透射 transmission / 各向异性 anisotropy），最严谨。

### 材质 → 光学签名对照表（从画面里读签名，反推材质）

| 材质类 | 可见的光学签名 |
|:--|:--|
| **金属** | 强镜面反射、能映出环境轮廓；有色金属（金/铜）高光带色；拉丝金属＝**各向异性**长条状高光；几乎无漫反射底色 |
| **玻璃 / 透明体** | 透射 + 折射（背后物体被扭曲/错位）；边缘与掠射角处强反射（**菲涅尔**）；高光锐利 |
| **水 / 液体 / 湿表面** | 锐利明亮镜面高光 + 折射；**湿润使表面变暗、高光更集中**（干→湿的判据） |
| **皮肤** | **次表面散射**（薄处如耳廓、手指逆光透出暖红）；柔和漫反射 + 油光形成的小而散的高光 |
| **布料-哑光（棉/麻）** | 漫反射为主、几乎无高光、可见纤维织纹 |
| **布料-丝/缎** | **各向异性**柔光带，高光随褶皱走向滑动 |
| **绒 / 天鹅绒** | 边缘回光（asperity scattering），轮廓和褶皱转折处发亮 |
| **木** | 漫反射 + 方向性木纹；上漆木材多一层清漆镜面高光 |
| **塑料** | 均匀漫反射 + 单一集中高光；可做亮面也可做哑面 |
| **陶瓷 / 上釉** | 光滑漫反射底 + 锐利清漆高光（"釉感"） |
| **纸 / 未涂布材** | 漫反射、低反光、可见纤维或颗粒；糙面无高光 |

### 表面属性维度（逐个主要表面打分）

光滑↔粗糙 · 反光↔哑光 · 透明↔半透↔不透 · 干↔湿 · 崭新↔磨损做旧。

### 纹理

给方向（木纹横纵、拉丝走向）、尺度（粗颗粒/细腻）、规律性（规则织纹/随机斑驳）。

### 取证 + 精度

- 逐个**主要表面**给材质判断 + 依据，不要只说一句"材质丰富"。
- 取证示范：`主体面部判为湿润/油润皮肤。依据：① T 区高光呈锐利小亮点（强镜面成分→有油光/湿度）；② 鼻翼、耳廓逆光处透出暖红（次表面散射→皮肤特征）；③ 整体仍以柔和漫反射为底。` ；`手持容器判为磨砂玻璃。依据：高光被柔化扩散（非锐利镜面→表面有微观粗糙度）、能见到内容物轮廓但模糊（半透射）、边缘有菲涅尔亮边。`
