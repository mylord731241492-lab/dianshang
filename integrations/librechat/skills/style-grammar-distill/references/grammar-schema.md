## Schema

```
# 风格语法：<作者/体系名> · 基于 N 张语料

## 0. 语料充分性判断
- 语料构成（N 张，覆盖哪些品类/客户/命题）
- 多样性评估：够不够分离 brief/签名？最大盲区是什么？
- 本份语法可支撑的最强结论强度（签名级 / 规则级 / 仅线索级）

## 1. 签名层（认证通过的恒定原则）
每条：
- 签名陈述（写在它的恰当抽象高度上）
- 抽象高度（表面特征级 / 原则级 / 立场级——越低越具体）
- 覆盖度（N/M 张，跨几个不同命题）
- 收敛证据（哪些维度 × 哪些图共同佐证）
- 置信度（高/中/低）
- confound 状态（已与类型默认/brief 分离 / 待竞品集确认 / 语料盲区）

## 2. 条件规则层（goal→choice 映射，作者的适应逻辑）
每条：
- 触发条件（当目标/品类/定位 = X）
- 作者的选择（→ 则倾向 Y）
- 服务的目标（为什么——供下游从目标反推）
- 覆盖度 + 置信度

## 3. 恒定–变量–例外台账
- 恒定项：清单（指向签名层）
- 自由变量：作者不在意/外包/随机的维度（下游可自由发挥的空间）
- 例外/破例：逐条记录破例点 + "为何在此破例"的解读（规则背后的判断）

## 4. 风格语言摘要（一段话）
- 用一段连贯文字概括"这位作者搭画面的底层逻辑"——给人快速抓住,且点明其核心张力

## 5. 污染与存疑清单
- 哪些"签名"可能是 brief/类型默认未分离干净
- 语料盲区（缺哪类命题导致哪条结论悬空）
- 抽象高度存疑（哪条爬得可能过高或过低）
- 下一步该补什么语料来证伪/坐实
```

## 完整示例（虚构 4 张语料）

> 沿用 aggregation-method.md 里那位"暗黑产品 KV"作者的设想 4 张跨命题作品。**纯属虚构,仅演示语法的颗粒度与诚实标注**,不对应任何真实摄影师/AD。

**# 风格语法:某产品视觉作者 · 基于 4 张语料**

**## 0. 语料充分性判断**
4 张:图1 暗黑膏霜 KV(高端护肤)、图2 高明度护肤(清新护肤)、图3 暗黑彩妆(另一高端客户)、图4 产品使用场景(中端)。多样性:跨 3 个客户、2 个价位带、2 种色调键——**勉强够**分离部分 brief/签名,但**全是产品品类、无人像/无叙事广告**,故关于"人物调度"的签名无法提炼。最大盲区:缺"暖色/高饱和"命题,无法确认"去彩"是签名还是偏好。本份语法**可支撑到"签名级",但限于产品静物范畴**。

**## 1. 签名层**

- **S1 · 环境剥离,孤立单一主体**
  - 抽象高度:**原则级**(非表面"黑底",而是"去语境化")
  - 覆盖度:4/4,跨全部 3 个客户
  - 收敛证据:背景 void(4/4)× 极端负空间(4/4)× 单一英雄主体(4/4)× 无生活场景符号(4/4)——四维收敛
  - 置信度:**高**
  - confound:已与 brief 分离(跨不同客户/价位均成立);非类型默认(部分竞品会用场景图,作者坚持剥离)
- **S2 · 单一色调世界推到极致,从不折中**
  - 抽象高度:原则级(注意:**不是**"暗调"——图2 证伪了暗调)
  - 覆盖度:4/4
  - 收敛证据:每张的色调键都被推满(暗的极暗/亮的极亮),无中间地带的"安全"处理
  - 置信度:高
  - confound:已分离(具体色调随定位变=见 R1,但"推到极致"这条恒定)
- **S3 · 恋物式单点触觉/工程英雄细节**
  - 抽象高度:原则级
  - 覆盖度:4/4(球关节/泵头/滚花/旋盖)
  - 收敛证据:每张都有且仅有一个被特写强调的触觉细节
  - 置信度:**中**(复现强,但"仅一个"可能是产品本身决定,需更多语料确认是作者编排还是产品自带)
- **S4 · 大面积柔光 + 高反差并存**
  - 抽象高度:表面特征级(偏低,较具体)
  - 覆盖度:3/4(图2 高明度处低反差)→ 实为协变,**降级**,见 R2

**## 2. 条件规则层（goal→choice）**

- **R1**:当**品牌定位 = 高端/科技/austere** → 倾向**暗调高反差 + 冷调去彩 void**;当**定位 = 清新/亲和** → 倾向**高明度低反差 + 白 void**。
  - 服务目标:用色调世界整体承载品牌气质,让"调性"在第一眼传达。
  - 覆盖度 4/4,置信度高。
- **R2**:光反差**协变于**色调键(暗调→高反差,高明度→低反差),不是独立恒定。
  - 服务目标:反差服从于既定色调世界的统一性。
  - 置信度中。
- **R3**:当**产品有可特写的机械/工程结构** → 必给它一个英雄特写位;当**产品造型平淡** → (语料未见,**留空待验**)。

**## 3. 恒定–变量–例外台账**
- **恒定项**:环境剥离(S1)、单一色调世界推满(S2)、单点触觉英雄(S3)。
- **自由变量**:具体背景色值(随定位变,非自由——属 R1)、构图对角/垂直的具体角度(各图不同,无规律→真自由,下游可发挥)。
- **例外**:图4 引入"一只手"——是 4 张里唯一让人入画的破例。解读:可能因"使用场景"命题强制需要人;**存疑**——这是作者对"使用类命题"的固定应对(则应升为规则 R4),还是一次性妥协?**需更多使用类命题的图来定。**

**## 4. 风格语言摘要**
这位作者的底层逻辑是**"器物纯粹主义"**:无论命题明暗,都把产品从一切语境里剥出来,孤立在无信息的 void 中(S1),配一个被推到极致、整体承载品牌气质的单一色调世界(S2/R1),并总要找到产品身上一个可被恋物式特写的触觉/工程细节(S3)。核心张力在于:他用极强的控制力制造"完美无尘"的器物感,而这种完美本身既是其高级感来源,也是其最大风险(易滑向"空洞高级/换个品牌也成立")。

**## 5. 污染与存疑清单**
- "去彩/冷调"无法确认是签名——语料缺暖色/高饱和命题,可能只是这 4 个 brief 恰好都偏冷。**需一张暖调命题证伪。**
- S3"仅一个触觉细节":可能是产品自带而非作者编排,**需更多产品形态各异的语料。**
- 图4 的"手"是规则还是例外,悬而未决(见台账)。
- 全语料无人像/叙事广告,**任何涉及人物调度、叙事的风格主张都不在本语法覆盖内**,下游 Stage B 不得越界生成这类内容。

---

> 注意这份语法如何处处区分"签名 / 条件规则 / 例外",如何标注每条的抽象高度与置信度,以及如何在结尾**主动暴露盲区**而非假装完整。这就是 Stage A 的价值:不是给一份"他喜欢什么"的清单,而是给一份**带不确定性标注、能让下游从目标反推的生成语法**——并诚实地圈出它还不知道的部分。
